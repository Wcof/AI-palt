# AI 对话前景层视觉优化说明

本次优化围绕首页对话工作区进行，目标是解决“对话框太小、前景泛白、抬头不明显”的问题。

## 调整内容

1. 对话区抬头放大
   - 会话标题从小字号提升为大字号主标题。
   - 默认标题调整为「极客光年安全智能」。
   - 标题颜色使用深蓝，避免与浅色背景混在一起。

2. 对话框尺寸增强
   - 输入框高度从 48px 提升到 76px。
   - 输入框增加浅蓝边框、白蓝渐变底色和聚焦态。
   - 发送按钮、图片按钮同步放大到 48px。

3. 前景层增加颜色区分
   - 主内容区增加浅蓝径向底纹。
   - 消息气泡从纯白改为白蓝渐变卡片。
   - 用户消息保持蓝色渐变，不再使用黑色风格。
   - 欢迎页标题区、推荐问题卡片、底部提示条增加蓝色前景层次。

4. 深度思考样式优化
   - 思考卡片改为蓝色主题链路卡片。
   - 步骤序号使用蓝色渐变圆点，整体与安全生产蓝色风格一致。
