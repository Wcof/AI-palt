# 页面 7.2：应用管理 `/workbench/apps`

说明：个人中心 `/profile` 与账户设置 `/settings` 为外部统一认证平台入口，本平台不承载对应页面，仅提供跳转。

## 模块一：角色与权限总览

| 功能区块 | 未登录访客 | 已登录用户 | 管理员 |
|---|---|---|---|
| 应用列表 | 可见 | 可见 | 可见 |
| 新增应用 | 按钮置灰，不可点击 | 可见且可操作 | 可见且可操作 |
| 编辑应用 | 按钮置灰，不可点击 | 可见且可操作 | 可见且可操作 |
| 删除应用 | 按钮置灰，不可点击 | 可见且可操作 | 可见且可操作 |
| 批量操作 | 按钮置灰，不可点击 | 可见且可操作 | 可见且可操作 |

## 模块二：正常流程

分支 A — 新增应用：
1. 用户点击“创建应用”。
2. 弹出创建应用弹窗。
3. 用户填写应用名称。
4. 用户填写应用描述。
5. 用户选择状态。
6. 用户点击“提交”。
7. 列表新增应用记录。

分支 B — 编辑应用：
1. 用户点击“编辑”。
2. 弹出编辑弹窗并回填数据。
3. 用户修改字段。
4. 用户点击“保存”。
5. 列表更新。

分支 C — 批量操作：
1. 用户勾选多个应用。
2. 用户点击“批量启用/禁用/删除”。
3. 列表更新状态或删除记录。

## 模块三：交互与字段说明

### 功能区块：创建/编辑弹窗
#### 元素：应用名称输入框
- 字段来源：用户输入
- 字段约束：1~50 字，支持中英文、数字、空格及常用符号，首尾空格自动去除；必填；服务端参数化处理防注入
- 鼠标悬停时的界面变化：无悬停效果
- 点击/聚焦时的界面变化：获得输入焦点
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录

#### 元素：应用描述输入框
- 字段来源：用户输入
- 字段约束：0~200 字，支持中英文、数字、空格及常用符号，首尾空格自动去除；服务端参数化处理防注入
- 鼠标悬停时的界面变化：无悬停效果
- 点击/聚焦时的界面变化：获得输入焦点
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录

#### 元素：状态选择
- 字段来源：用户选择
- 字段约束：启用/禁用
- 鼠标悬停时的界面变化：无悬停效果
- 点击/聚焦时的界面变化：切换状态
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录

#### 元素：提交按钮
- 字段来源：系统
- 字段约束：名称必填
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：提交表单
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：名称为空或未登录

## 模块四：异常流程与规则

1. 数据加载中：列表区域显示占位骨架屏，结构与真实列表一致。
2. 列表/页面无数据时：显示“暂无应用”。
3. 请求失败/网络不可用时：弹出提示“网络异常，请稍后重试”。
4. 未登录用户触发需登录的操作：提示“请先登录”并跳转统一认证平台，登录后回跳本页。
5. 表单/输入校验不通过时：提示“请完善必填信息”，对应字段高亮。
6. 地址栏参数非法时：本页面不涉及。

## 模块五：验收标准

验收项 1：创建应用
前提条件：已登录
执行操作：填写名称并提交
预期结果：列表新增应用

验收项 2：创建校验
前提条件：已登录
执行操作：名称为空点击提交
预期结果：按钮置灰或提示文案

验收项 3：编辑应用
前提条件：已登录
执行操作：编辑并保存
预期结果：列表更新

验收项 4：删除应用
前提条件：已登录
执行操作：点击删除
预期结果：记录移除

验收项 5：批量启用
前提条件：已登录且勾选多条
执行操作：点击批量启用
预期结果：状态更新

验收项 6：批量删除
前提条件：已登录且勾选多条
执行操作：点击批量删除
预期结果：记录移除

验收项 7：未登录限制
前提条件：未登录
执行操作：点击创建应用
预期结果：按钮不可点击

验收项 8：搜索过滤
前提条件：列表有数据
执行操作：输入关键词
预期结果：列表过滤

## 模块六：局部非功能性需求

- 弹窗打开关闭无页面刷新。
- 批量操作需要二次确认，提示“确认对所选记录执行批量操作？”。
