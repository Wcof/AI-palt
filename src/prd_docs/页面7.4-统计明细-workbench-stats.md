# 页面 7.4：统计明细 `/workbench/stats`

说明：个人中心 `/profile` 与账户设置 `/settings` 为外部统一认证平台入口，本平台不承载对应页面，仅提供跳转。

## 模块一：角色与权限总览

| 功能区块 | 未登录访客 | 已登录用户 | 管理员 |
|---|---|---|---|
| 统计筛选 | 可见但不可操作（需登录后使用） | 可见且可操作 | 可见且可操作 |
| 趋势图与统计卡片 | 可见但不可操作（需登录后使用） | 可见且可操作 | 可见且可操作 |
| 导出 CSV | 可见但不可操作（需登录后使用） | 可见且可操作 | 可见且可操作 |
| 模拟数据按钮 | 可见但不可操作（需登录后使用） | 可见且可操作 | 可见且可操作 |

## 模块二：正常流程

1. 用户进入统计明细页面。
2. 用户选择时间范围。
3. 用户选择应用筛选。
4. 用户选择模型筛选。
5. 系统刷新趋势图与统计卡片。
6. 用户点击“生成 50 条模拟数据”。
7. 列表与图表刷新。
8. 用户点击“导出 CSV”。

## 模块三：交互与字段说明

### 功能区块：筛选区
#### 元素：时间范围选择
- 字段来源：用户选择
- 字段约束：日期格式合法
- 鼠标悬停时的界面变化：无悬停效果
- 点击/聚焦时的界面变化：打开日期选择
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录

#### 元素：应用筛选
- 字段来源：系统
- 字段约束：单选
- 鼠标悬停时的界面变化：无悬停效果
- 点击/聚焦时的界面变化：切换筛选并刷新
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录

#### 元素：模型筛选
- 字段来源：系统
- 字段约束：单选
- 鼠标悬停时的界面变化：无悬停效果
- 点击/聚焦时的界面变化：切换筛选并刷新
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录

### 功能区块：操作按钮
#### 元素：生成模拟数据按钮
- 字段来源：系统
- 字段约束：需存在可用密钥
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：生成模拟数据并刷新
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录或无可用密钥

#### 元素：实时模拟按钮
- 字段来源：系统
- 字段约束：需存在可用密钥
- 鼠标悬停时的界面变化：按钮高亮

- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录或无可用密钥

#### 元素：导出 CSV 按钮
- 字段来源：系统
- 字段约束：需有数据
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：下载 CSV
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录或无数据

## 模块四：异常流程与规则

1. 数据加载中：图表与列表显示占位骨架屏，操作禁用。
2. 列表/页面无数据时：显示“暂无数据”。
3. 请求失败/网络不可用时：弹出提示“网络异常，请稍后重试”。
4. 未登录用户触发需登录的操作：提示“请先登录”并跳转统一认证平台，登录后回跳本页。
5. 表单/输入校验不通过时：提示“请完善必填信息”。
6. 地址栏参数非法时：本页面不涉及。

## 模块五：验收标准

验收项 1：时间筛选
前提条件：已登录
执行操作：选择“近 7 天”
预期结果：图表与列表刷新

验收项 2：应用筛选
前提条件：有应用数据
执行操作：选择应用
预期结果：图表与列表刷新

验收项 3：模型筛选
前提条件：有模型数据
执行操作：选择模型
预期结果：图表与列表刷新

验收项 4：生成模拟数据
前提条件：有可用密钥
执行操作：点击“生成 50 条模拟数据”
预期结果：列表新增记录

验收项 5：实时模拟
前提条件：有可用密钥
执行操作：点击“开启实时模拟”
预期结果：按钮状态变化并持续新增记录

验收项 6：导出 CSV
前提条件：有统计数据
执行操作：点击“导出 CSV”
预期结果：下载 CSV 文件

验收项 7：无数据空态
前提条件：无统计数据
执行操作：进入页面
预期结果：显示“暂无数据”

验收项 8：无密钥限制
前提条件：无可用密钥
执行操作：点击模拟按钮
预期结果：按钮置灰不可点击
