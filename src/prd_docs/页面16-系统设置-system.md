# 页面 16：系统设置 `/system`

## 模块一：角色与权限总览

| 功能区块 | 未登录访客 | 已登录用户 | 管理员 |
|---|---|---|---|
| 品牌配置 | 不可见 | 可见且可操作 | 可见且可操作 |
| 推荐问题配置 | 不可见 | 可见且可操作 | 可见且可操作 |

## 模块二：正常流程

分支 A — 切换标签页：
1. 用户进入系统设置页面。
2. 系统默认显示"品牌配置"标签页。
3. 用户点击"问题配置"标签页。
4. 页面切换至问题配置内容。

分支 B — 配置品牌信息：
1. 用户在"品牌配置"标签页中。
2. 用户输入系统名称。
3. 用户输入页签抬头。
4. 用户输入页签 Icon URL 或点击"上传页签 Icon"上传文件。
5. 用户输入项目 Logo URL 或点击"上传项目 Logo"上传文件。
6. 系统实时保存所有修改至本地存储。
7. 用户刷新页面，配置仍保留。

分支 C — 添加推荐问题：
1. 用户在"问题配置"标签页中。
2. 用户点击"添加问题"按钮。
3. 弹出"添加推荐问题"对话框。
4. 用户输入问题标题。
5. 用户输入问题描述。
6. 用户输入问题内容。
7. 用户选择问题图标。
8. 用户点击"保存"按钮。
9. 对话框关闭，新问题添加至列表。

分支 D — 编辑推荐问题：
1. 用户在"问题配置"标签页中。
2. 用户在问题卡片上点击编辑按钮。
3. 弹出"编辑推荐问题"对话框，预填当前问题信息。
4. 用户修改问题信息。
5. 用户点击"保存"按钮。
6. 对话框关闭，问题信息更新。

分支 E — 删除推荐问题：
1. 用户在"问题配置"标签页中。
2. 用户在问题卡片上点击删除按钮。
3. 问题立即从列表中删除。

分支 F — 文件上传：
1. 用户点击"上传页签 Icon"或"上传项目 Logo"按钮。
2. 系统打开文件选择对话框。
3. 用户选择图片文件。
4. 系统将文件转换为 Data URL 并保存。
5. 页面显示"已设置"状态。

## 模块四：异常流程与规则

1. 数据加载中：本页面不涉及。
2. 列表/页面无数据时：问题列表为空时显示"暂无推荐问题"提示。
3. 请求失败/网络不可用时：本页面不涉及（所有数据本地存储）。
4. 未登录用户触发需登录的操作：本页面不涉及。
5. 表单/输入校验不通过时：保存按钮置灰，不可点击；必填项为空时无法保存。
6. 地址栏参数非法时：本页面不涉及。
7. 文件上传失败时：显示提示"文件上传失败，请重试"。
8. 达到问题上限时：添加问题按钮置灰，显示"最多只能添加 4 个问题"提示。

## 模块五：验收标准

验收项 1：切换标签页
前提条件：进入系统设置页面
执行操作：点击"问题配置"标签页
预期结果：页面切换至问题配置内容

验收项 2：配置系统名称
前提条件：在品牌配置标签页
执行操作：输入系统名称并刷新页面
预期结果：系统名称保留，本地存储生效

验收项 3：上传页签 Icon
前提条件：在品牌配置标签页
执行操作：点击"上传页签 Icon"并选择图片文件
预期结果：文件转换为 Data URL，显示"已设置页签 Icon"

验收项 4：添加推荐问题
前提条件：在问题配置标签页，问题数少于 4 个
执行操作：点击"添加问题"，填写信息并保存
预期结果：新问题添加至列表

验收项 5：编辑推荐问题
前提条件：问题列表中存在问题
执行操作：点击问题的编辑按钮，修改信息并保存
预期结果：问题信息更新

验收项 6：删除推荐问题
前提条件：问题列表中存在问题
执行操作：点击问题的删除按钮
预期结果：问题从列表中删除

验收项 7：问题上限限制
前提条件：已添加 4 个问题
执行操作：查看"添加问题"按钮状态
预期结果：按钮置灰，显示"最多只能添加 4 个问题"提示

验收项 8：必填项校验
前提条件：在添加/编辑问题对话框中
执行操作：不填写问题标题或内容，点击保存
预期结果：保存按钮不可点击或显示错误提示

## 模块六：局部非功能性需求
