# 页面 7.3：API 密钥管理 `/workbench/keys`

说明：个人中心 `/profile` 与账户设置 `/settings` 为外部统一认证平台入口，本平台不承载对应页面，仅提供跳转。

## 模块一：角色与权限总览

| 功能区块 | 未登录访客 | 已登录用户 | 管理员 |
|---|---|---|---|
| 密钥列表 | 可见 | 可见 | 可见 |
| 创建密钥 | 按钮置灰，不可点击 | 可见且可操作 | 可见且可操作 |
| 删除密钥 | 按钮置灰，不可点击 | 可见且可操作 | 可见且可操作 |
| 查看密钥 | 按钮置灰，不可点击 | 可见且可操作 | 可见且可操作 |

## 模块二：正常流程

分支 A — 创建密钥：
1. 用户点击“创建 API 密钥”。
2. 弹出创建弹窗。
3. 用户选择所属应用。
4. 用户点击“提交”。
5. 列表新增密钥记录（名称与状态自动生成）。

分支 B — 查看密钥：
1. 用户点击“查看 API Key”。
2. 弹出密钥详情弹窗。
3. 用户点击“关闭”。

分支 C — 删除密钥：
1. 用户勾选密钥。
2. 用户点击“删除”。
3. 记录从列表移除。

## 模块三：交互与字段说明

### 功能区块：创建弹窗
#### 元素：所属应用选择
- 字段来源：系统
- 字段约束：必选
- 鼠标悬停时的界面变化：无悬停效果
- 点击/聚焦时的界面变化：打开下拉并选择
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录

#### 元素：提交按钮
- 字段来源：系统
- 字段约束：应用必选
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：提交表单
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：字段未填或未登录

### 功能区块：列表操作
#### 元素：查看按钮
- 字段来源：系统
- 字段约束：不适用（非输入项）
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：打开详情弹窗
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录

#### 元素：删除按钮
- 字段来源：系统
- 字段约束：需勾选
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：删除记录
- 角色差异：未登录不可操作
- 置灰/不可点击触发条件：未登录或未勾选

## 模块四：异常流程与规则

1. 数据加载中：列表区域显示占位骨架屏，结构与真实列表一致。
2. 列表/页面无数据时：显示“暂无密钥”。
3. 请求失败/网络不可用时：弹出提示“网络异常，请稍后重试”。
4. 未登录用户触发需登录的操作：提示“请先登录”并跳转统一认证平台，登录后回跳本页。
5. 表单/输入校验不通过时：提示“请选择关联应用”，对应字段高亮。
6. 地址栏参数非法时：本页面不涉及。

## 模块五：验收标准

验收项 1：创建密钥
前提条件：已登录
执行操作：选择应用并提交
预期结果：列表新增密钥

验收项 2：创建校验
前提条件：已登录
执行操作：未选择应用点击提交
预期结果：按钮置灰或提示“请选择关联应用”

验收项 3：查看密钥
前提条件：已登录且列表有数据
执行操作：点击“查看 API Key”
预期结果：弹出密钥详情

验收项 4：删除密钥
前提条件：已登录且勾选密钥
执行操作：点击“删除”
预期结果：记录移除

验收项 5：未登录限制
前提条件：未登录
执行操作：点击创建密钥
预期结果：按钮不可点击

验收项 6：搜索过滤
前提条件：列表有数据
执行操作：输入关键词
预期结果：列表过滤

验收项 7：批量选择
前提条件：列表有数据
执行操作：点击全选
预期结果：选中所有密钥

验收项 8：查看弹窗关闭
前提条件：已打开详情弹窗
执行操作：点击“关闭”
预期结果：弹窗关闭

## 模块六：局部非功能性需求

- 密钥明文仅在详情弹窗中展示。
