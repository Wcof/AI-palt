## 模块三：正常流程

分支 A — 选择知识库与预览：
1. 用户进入页面，左侧展示知识库列表（仅「安全生产知识库」一个条目）。
2. 用户点击该知识库条目。
3. 右侧上下文切换为该知识库。
4. 文档面板弹出并展示该知识库的文档列表。

分支 B — 对话体验：
1. 用户在对话输入框输入问题。
2. 用户点击"发送"。
3. 对话区展示回答。

分支 C — 关闭文档面板：
1. 用户点击文档面板右上角"关闭"。
2. 文档面板关闭。

分支 D — URL 自动选中：
1. 用户通过 `?kb=kb-safety-production` 进入页面。
2. 系统自动选中安全生产知识库并打开文档面板。

## 模块四：交互与字段说明

### 功能区块：知识库列表
#### 元素：知识库条目
- 字段来源：系统
- 字段约束：单选；当前列表仅含「安全生产知识库」一个条目
- 鼠标悬停时的界面变化：条目高亮
- 点击/聚焦时的界面变化：切换为该知识库并打开文档面板
- 角色差异：无
- 置灰/不可点击触发条件：无

### 功能区块：对话区
#### 元素：对话输入框
- 字段来源：用户输入
- 字段约束：最多 5000 字，支持中英文、数字、空格及常用符号；服务端参数化处理防注入
- 鼠标悬停时的界面变化：无悬停效果
- 点击/聚焦时的界面变化：获得输入焦点
- 角色差异：无
- 置灰/不可点击触发条件：无

#### 元素：发送按钮
- 字段来源：系统
- 字段约束：输入为空不可发送
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：发送问题并展示回答
- 角色差异：无
- 置灰/不可点击触发条件：输入为空

#### 元素：清空对话按钮
- 字段来源：系统
- 字段约束：不适用（非输入项）
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：清空对话内容
- 角色差异：无
- 置灰/不可点击触发条件：无

### 功能区块：文档面板
#### 元素：关闭按钮
- 字段来源：系统

- 点击/聚焦时的界面变化：关闭文档面板
- 角色差异：无

## 模块五：异常流程与规则

1. 数据加载中：列表区域显示占位骨架屏，结构与真实列表一致。
2. 列表/页面无数据时：显示"暂无知识库"。
3. 请求失败/网络不可用时：弹出提示"网络异常，请稍后重试"。
4. 未登录用户触发需登录的操作：本页面不涉及。
5. 表单/输入校验不通过时：发送按钮置灰不可点击，无提示文案。
6. 地址栏参数非法时：忽略非法参数，不自动选中知识库。
