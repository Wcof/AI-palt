# 页面 12：AI 智能体在线体验 `/experience/agent`

说明：个人中心 `/profile` 与账户设置 `/settings` 为外部统一认证平台入口，本平台不承载对应页面，仅提供跳转。

> **【2026-03-03 版本调整说明】**
> AI 智能体调整：包含「EHS 智能体」「NL2SQL」「隐患识别」三个智能体。其中 NL2SQL 和隐患识别从 AI 技能（MCP）迁移至此，并采用对话框交互形式。其余智能体（Agent 套件、文档助手、对话问数、多模态推理、代码助手、对话机器人、提示词库）已全部移除。

## 模块一：角色与权限总览

|---|---|---|---|
| 智能体列表 | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） |
| 智能体实验台 | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） |
| 能力预览弹窗 | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） |

## 模块二：当前智能体清单

| 智能体名称 | API 名称 | 简介 | 交互方式 |
|---|---|---|---|
| EHS 智能体 | ehs-llm-api | 面向安全生产的智能问答与合规助手，问一句就给可执行建议。 | 实验台模式 |
| NL2SQL | nl2sql-llm-api | 将自然语言问题自动转换为 SQL 查询语句，支持安全生产数据库智能问数，无需编写代码即可完成数据分析。 | 对话框模式 |
| 隐患识别 | hazard-detection-llm-api | 基于 AI 视觉与语义分析，自动识别作业现场安全隐患，生成隐患报告与整改建议，助力安全风险管控。 | 对话框模式（仅支持图片上传） |

## 模块三：正常流程

分支 A — 选择智能体（EHS 智能体）：
1. 用户进入页面，系统加载智能体列表（共 3 个：EHS 智能体、NL2SQL、隐患识别）。
2. 用户点击「EHS 智能体」条目。
3. 右侧实验台显示"当前智能体：EHS 智能体"，展示实验台界面（系统提示词、参数配置等）。

分支 B — 选择智能体（NL2SQL）：
1. 用户点击「NL2SQL」条目。
2. 右侧切换为对话框界面。
3. 用户在输入框输入自然语言问题（如"查询本月安全事故次数"）。
4. 用户点击"发送"。
5. 系统返回 SQL 查询结果并在对话框中展示。

分支 C — 选择智能体（隐患识别）：
1. 用户点击「隐患识别」条目。
2. 右侧切换为对话框界面（仅支持图片上传）。
3. 用户点击"上传图片"按钮，选择现场作业图片。
4. 用户点击"发送"。
5. 系统分析图片并返回隐患识别报告（包含隐患类型、位置、风险等级、整改建议等）。

分支 D — 预览智能体：
1. 用户在列表中双击任一智能体条目。
2. 弹出"能力预览"弹窗。
3. 用户查看智能体基础信息、标签与适用场景。

分支 E — 运行智能体（EHS 智能体实验台模式）：
1. 用户在"系统提示词"输入框输入内容。
2. 用户在"用户任务"输入框输入内容。
3. 用户调整 Temperature 或 Max Tokens。
4. 用户点击"运行智能体"。
5. 输出结果区域展示模拟响应内容。

## 模块四：交互与字段说明

### 功能区块：智能体列表
#### 元素：智能体条目
- 字段来源：系统
- 字段约束：单选；当前包含「EHS 智能体」「NL2SQL」「隐患识别」三个条目

- 点击/聚焦时的界面变化：右侧切换为对应智能体界面（实验台模式或对话框模式）；双击打开能力预览弹窗

- 置灰/不可点击触发条件：无

### 功能区块：能力预览弹窗

## 模块五：异常流程与规则

1. 数据加载中：列表区域显示占位骨架屏，结构与真实列表一致。
2. 列表/页面无数据时：显示"暂无智能体"。
3. 请求失败/网络不可用时：弹出提示"网络异常，请稍后重试"。
4. 未登录用户触发需登录的操作：本页面不涉及。
5. 表单/输入校验不通过时：本页面不涉及。
6. 地址栏参数非法时：忽略非法参数，不自动选中智能体。

## 模块七：局部非功能性需求

- 实验台切换智能体时不刷新页面。

- 图片上传支持拖拽操作。
