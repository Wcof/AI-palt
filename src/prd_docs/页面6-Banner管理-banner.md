# 页面 6：Banner 管理 `/banner`

说明：个人中心 `/profile` 与账户设置 `/settings` 为外部统一认证平台入口，本平台不承载对应页面，仅提供跳转。

## 模块一：角色与权限总览

| 功能区块 | 未登录访客 | 已登录用户 | 管理员 |
|---|---|---|---|
| Banner 列表 | 不可见（访问跳转登录） | 可见但不可操作（需管理员） | 可见且可操作 |
| 新增/编辑表单 | 不可见（访问跳转登录） | 可见但不可操作（需管理员） | 可见且可操作 |
| 排序/启用/停用/删除 | 不可见（访问跳转登录） | 可见但不可操作（需管理员） | 可见且可操作 |

## 模块二：正常流程

分支 A — 新增 Banner：
1. 用户点击“选择图片”。
2. 用户选择图片文件。
3. 表单显示“已选择图片”。
4. 用户填写“Banner 标题”。
5. 用户填写“Banner 描述”。
6. 用户填写“跳转链接（可选）”。
7. 用户点击“新增 Banner”。
8. 列表新增一条 Banner 记录并出现在顶部。

分支 B — 编辑 Banner：
1. 用户点击列表中的“编辑”。
2. 表单进入编辑状态并回填数据。
3. 用户修改标题/描述/链接或重新选择图片。
4. 用户点击“保存修改”。
5. 列表内容更新。

分支 C — 调整顺序：
1. 用户点击“上移”按钮。
2. 列表顺序立即调整。
3. 用户点击“下移”按钮。
4. 列表顺序立即调整。

分支 D — 启用/停用：
1. 用户点击“已启用/已停用”按钮。
2. 状态切换并实时显示。

分支 E — 删除：
1. 用户点击“删除”。
2. 该 Banner 从列表移除。

## 模块三：交互与字段说明

### 功能区块：新增/编辑表单
#### 元素：Banner 标题输入框
- 字段来源：用户输入
- 字段约束：1~50 字，支持中英文、数字、空格及常用符号，首尾空格自动去除；必填；服务端参数化处理防注入
- 鼠标悬停时的界面变化：无悬停效果
- 点击/聚焦时的界面变化：获得输入焦点
- 角色差异：无
- 置灰/不可点击触发条件：无

## 模块四：异常流程与规则

1. 数据加载中：列表区域显示占位骨架屏，结构与真实列表一致。
2. 列表/页面无数据时：显示“暂无 Banner，可在上方新增”。
3. 请求失败/网络不可用时：弹出提示“网络异常，请稍后重试”。
4. 未登录用户触发需登录的操作：提示“请先登录”并跳转统一认证平台；非管理员提示“无权限，请联系管理员”。
5. 表单/输入校验不通过时：提示“请完善必填信息”，对应字段高亮。
6. 地址栏参数非法时：本页面不涉及。

## 模块五：验收标准

验收项 1：新增 Banner
前提条件：标题、描述、图片均已填写
执行操作：点击“新增 Banner”
预期结果：列表新增一条记录并在顶部展示

验收项 2：编辑 Banner
前提条件：列表存在 Banner
执行操作：点击“编辑”并保存
预期结果：列表内容更新

验收项 3：删除 Banner
前提条件：列表存在 Banner
执行操作：点击“删除”
预期结果：该 Banner 从列表移除

验收项 4：调整顺序
前提条件：列表至少两条
执行操作：点击“上移/下移”
预期结果：顺序变化

验收项 5：启用/停用
前提条件：列表存在 Banner
执行操作：点击“已启用/已停用”
预期结果：状态切换

验收项 6：表单校验
前提条件：标题为空
执行操作：点击“新增 Banner”
预期结果：按钮置灰或提示文案

验收项 7：预览一致性
前提条件：已选图片并填写标题/描述
执行操作：查看预览区
预期结果：预览样式与首页 Banner 一致

验收项 8：链接可为空
前提条件：链接为空
执行操作：新增 Banner
预期结果：列表展示链接为“-”
