# 页面 7：开发工作台 `/workbench`

说明：个人中心 `/profile` 与账户设置 `/settings` 为外部统一认证平台入口，本平台不承载对应页面，仅提供跳转。

## 模块一：角色与权限总览

| 功能区块 | 未登录访客 | 已登录用户 | 管理员 |
|---|---|---|---|
| 工作台导航 | 可见但不可操作（需登录后使用） | 可见且可操作 | 可见且可操作 |
| 子模块内容 | 可见但不可操作（需登录后使用） | 可见且可操作 | 可见且可操作 |

## 模块二：正常流程

分支 A — 已登录用户：
1. 用户进入 `/workbench`。
2. 系统自动跳转到订阅列表。
3. 用户点击“应用管理”。
4. 页面切换到应用管理页。

分支 B — 未登录访客：
1. 用户访问 `/workbench`。
2. 页面显示内容但不提供操作，点击任一操作提示登录并跳转统一认证平台。

## 模块三：交互与字段说明

### 功能区块：工作台导航
#### 元素：导航 Tab
- 字段来源：系统
- 字段约束：不适用（非输入项）
- 鼠标悬停时的界面变化：Tab 高亮
- 点击/聚焦时的界面变化：切换子页面
- 角色差异：未登录不可操作，触发操作提示登录并跳转统一认证平台。
- 置灰/不可点击触发条件：未登录

## 模块四：异常流程与规则

1. 数据加载中：显示占位骨架屏，导航不可操作。
2. 列表/页面无数据时：各子模块显示对应空态。
3. 请求失败/网络不可用时：弹出提示“网络异常，请稍后重试”。
4. 未登录用户触发需登录的操作：提示“请先登录”并跳转统一认证平台，登录后回跳本页。
5. 表单/输入校验不通过时：本页面不涉及。
6. 地址栏参数非法时：跳转默认子模块。

## 模块五：验收标准

验收项 1：默认路由
前提条件：已登录
执行操作：访问 `/workbench`
预期结果：跳转订阅列表

验收项 2：Tab 切换
前提条件：已登录
执行操作：点击“应用管理”
预期结果：进入应用管理页

验收项 3：未登录访问
前提条件：未登录
执行操作：访问 `/workbench`
预期结果：提示登录并跳转统一认证平台，登录后回跳当前页面。

验收项 4：导航高亮
前提条件：进入任一子模块
执行操作：观察导航
预期结果：当前 Tab 高亮

验收项 5：URL 同步
前提条件：进入任一子模块
执行操作：刷新页面
预期结果：停留在当前子模块

## 模块六：局部非功能性需求

- Tab 切换无页面刷新。
