# 页面 4：能力详情 `/ability/:id`

说明：个人中心 `/profile` 与账户设置 `/settings` 为外部统一认证平台入口，本平台不承载对应页面，仅提供跳转。

## 模块一：角色与权限总览

| 功能区块 | 未登录访客 | 已登录用户 | 管理员 |
|---|---|---|---|
| 能力基础信息 | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） |
| 标签与版本信息 | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） |
| 接口示例 | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） |
| 在线体验按钮（仅知识库） | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） |

## 模块二：正常流程

分支 A — 浏览能力：
1. 用户进入能力详情页。
2. 用户查看能力名称、API 名称与摘要。
3. 用户查看标签、价格、热度与版本信息。

分支 B — 查看相似能力：
1. 用户点击“查看相似能力”。
2. 页面跳转安全生产中心并带入标签筛选。

分支 C — 在线体验（知识库能力）：
1. 用户点击“在线体验”。
2. 页面跳转到 `/experience?kb=能力ID`。

## 模块三：交互与字段说明

### 功能区块：操作按钮
#### 元素：返回列表按钮
- 字段来源：系统
- 字段约束：不适用（非输入项）
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：返回安全生产中心页面
- 角色差异：无
- 置灰/不可点击触发条件：无

#### 元素：查看相似能力按钮
- 字段来源：系统
- 字段约束：不适用（非输入项）
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：跳转安全生产中心并带标签参数
- 角色差异：无
- 置灰/不可点击触发条件：无

#### 元素：在线体验按钮
- 字段来源：系统
- 字段约束：仅知识库能力显示
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：跳转在线体验页
- 角色差异：无
- 置灰/不可点击触发条件：非知识库能力不展示

## 模块四：异常流程与规则

1. 数据加载中：页面显示占位骨架屏。
2. 列表/页面无数据时：能力不存在时显示“能力不存在”。
3. 请求失败/网络不可用时：弹出提示“网络异常，请稍后重试”。
4. 未登录用户触发需登录的操作：本页面不涉及。
5. 表单/输入校验不通过时：本页面不涉及。
6. 地址栏参数非法时：显示“能力不存在”，提供返回入口。

## 模块五：验收标准

验收项 1：能力不存在
前提条件：访问不存在的能力 ID
执行操作：进入页面
预期结果：显示“能力不存在”和“返回安全生产中心”按钮

验收项 2：查看相似能力
前提条件：能力存在
执行操作：点击“查看相似能力”
预期结果：跳转安全生产中心并带标签筛选

验收项 3：在线体验
前提条件：能力类型为知识库
执行操作：点击“在线体验”
预期结果：跳转到 `/experience?kb=能力ID`

验收项 4：标签展示
前提条件：能力存在
执行操作：查看标签区域
预期结果：展示领域、价格、热度、版本标签

验收项 5：接口示例
前提条件：能力存在
执行操作：查看“接口调用示例”区域
预期结果：显示示例代码块

## 模块六：局部非功能性需求
