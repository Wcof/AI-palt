## 模块三：正常流程

分支 A — 通用浏览：
1. 用户进入页面，系统加载安全生产中心。
2. 用户滚动页面查看能力卡片（共 4 个：1 个知识库 + 3 个智能体）。
3. 用户点击任一能力卡片，进入能力详情页。

分支 B — 关键词搜索：
1. 用户在搜索输入框输入关键词。
2. 用户点击"搜索"。
3. 列表刷新并仅展示匹配能力。
4. 用户点击"清空"。
5. 关键词清空，列表恢复默认。

分支 C — 使用领域筛选：
1. 用户点击"领域"筛选项。
2. 用户选择"安全生产知识库"/"AI 智能体"/"AI 技能"之一。
3. 页面刷新列表并仅展示对应能力。
4. 用户点击"重置"按钮。
5. 页面恢复为全部能力列表。

分支 D — 使用分页：
1. 用户点击"下一页"按钮。
2. 页面切换到下一页列表。
3. 用户点击"上一页"按钮。
4. 页面切换到上一页列表。

## 模块五：异常流程与规则

1. 数据加载中：列表区域显示占位骨架屏，筛选条件不可修改。
2. 列表/页面无数据时：显示"暂无可展示能力"。
3. 请求失败/网络不可用时：弹出提示"网络异常，请稍后重试"。
4. 未登录用户触发需登录的操作：本页面不涉及。
5. 表单/输入校验不通过时：本页面不涉及。
6. 地址栏参数非法时：非法参数被忽略并使用默认筛选。
