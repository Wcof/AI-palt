# 页面 11：404 页面 `*`

说明：个人中心 `/profile` 与账户设置 `/settings` 为外部统一认证平台入口，本平台不承载对应页面，仅提供跳转。

## 模块一：角色与权限总览

| 功能区块 | 未登录访客 | 已登录用户 | 管理员 |
|---|---|---|---|
| 错误提示 | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） |
| 返回首页 | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） |
| 前往安全生产中心 | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） | 可见且可操作（公开访问，无角色限制） |

## 模块二：正常流程

1. 用户访问不存在地址。
2. 页面展示 404 提示。
3. 用户点击“返回首页”。
4. 页面跳转首页。
5. 用户返回后点击“前往安全生产中心”。
6. 页面跳转安全生产中心页面。

## 模块三：交互与字段说明

#### 元素：返回首页按钮
- 字段来源：系统
- 字段约束：不适用（非输入项）
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：跳转首页
- 角色差异：无
- 置灰/不可点击触发条件：无

#### 元素：前往安全生产中心按钮
- 字段来源：系统
- 字段约束：不适用（非输入项）
- 鼠标悬停时的界面变化：按钮高亮
- 点击/聚焦时的界面变化：跳转安全生产中心页面
- 角色差异：无
- 置灰/不可点击触发条件：无

## 模块四：异常流程与规则

1. 数据加载中：本页面不涉及。
2. 列表/页面无数据时：本页面不涉及。
3. 请求失败/网络不可用时：弹出提示“网络异常，请稍后重试”。
4. 未登录用户触发需登录的操作：本页面不涉及。
5. 表单/输入校验不通过时：本页面不涉及。
6. 地址栏参数非法时：本页面不涉及。

## 模块五：验收标准

验收项 1：404 展示
前提条件：访问不存在地址
执行操作：进入页面
预期结果：展示 404 提示

验收项 2：返回首页
前提条件：在 404 页面
执行操作：点击“返回首页”
预期结果：跳转首页

验收项 3：前往安全生产中心
前提条件：在 404 页面
执行操作：点击“前往安全生产中心”
预期结果：跳转安全生产中心页面

验收项 4：页面可访问
前提条件：任意用户
执行操作：访问不存在地址
预期结果：页面正常展示 404

验收项 5：按钮可点击
前提条件：在 404 页面
执行操作：点击任一按钮
预期结果：发生跳转

## 模块六：局部非功能性需求

- 页面加载 ≤ 2 秒。
