 第二期产品规划（纯功能方案）                                                  
                                                            
  📅 第一阶段（1-2个月）：核心体验打磨

  C 端客户端

  1. 场景化入口设计

  产品目标：让用户快速找到想要的功能，不是空白对话框

  功能描述：
  - 首页展示 5 大场景入口（卡片式）：
    - 特殊作业助手
    - 隐患排查助手
    - 应急处置助手
    - 合规检查助手
    - 安全培训助手
  - 每个场景入口显示：场景名称、简介、常用功能快捷入口
  - 点击进入后，对话框自动带入场景上下文

  ---
  2. 拍照识别隐患（MVP版）

  产品目标：现场拍照，AI 自动识别隐患并给出建议

  功能描述：
  - 用户拍照上传（支持相册选择或现场拍摄）
  - AI 识别隐患类型（人员违规、设备隐患、环境隐患）
  - 返回结果包含：
    - 隐患类型
    - 风险等级（一般/较大/重大/特别重大）
    - 整改建议（3-5条）
    - 依据法规
  - 支持单张识别
  - 识别结果可保存到历史记录

  ---
  3. 对话记录管理

  产品目标：像豆包一样的基础体验

  功能描述：
  - 多会话管理（按场景自动分类）
  - 历史记录列表（显示时间、场景、标题）
  - 搜索历史对话
  - 删除对话
  - 重命名对话标题

  ---
  4. 作业票智能填写（基础版）

  产品目标：减少重复劳动，快速生成作业票草稿

  功能描述：
  - 用户选择作业类型（动火、受限空间、高处作业等）
  - 通过对话方式描述作业内容
  - AI 自动生成作业票草稿（包含必填项）
  - 用户可编辑修改
  - 导出作业票（PDF/Word）

  ---
  技能平台

  1. API 市场（基础版）

  产品目标：开发者可以快速接入核心 AI 能力

    - 隐患识别 API
    - 知识库检索 API
    - NL2SQL API
  - API 文档页面（参数说明、示例）
  - 在线调试工具（输入参数，查看返回结果）

  产品目标：降低接入门槛

  功能描述：
  - 快速开始指南（5 分钟接入）
  - API 参考文档
  - 常见问题 FAQ
  - 联系支持

  ---
  3. API Key 管理

  产品目标：安全可控

  功能描述：
  - 创建 API Key

  - 配额管理（免费额度提醒）

  📅 第二阶段（2-3个月）：能力增强

  C 端客户端

  1. 拍照识别优化（增强版）

  产品目标：识别更准确，功能更完善

  功能优化点：

  a) 支持更多隐患类型
  - 人员违规：未戴安全帽、未穿反光衣、未系安全带、违规吸烟
  - 设备隐患：消防器材过期、电气线路裸露、防护罩缺失、设备漏油
  - 环境隐患：物料堆放不规范、通道堵塞、标识缺失、地面积水
  - 特殊作业隐患：动火作业无监护、受限空间无通风、高处作业无防护

  b) 批量识别
  - 一次上传多张照片（最多 20 张）
  - AI 自动汇总生成巡检报告
  - 报告包含：隐患统计、分类汇总、整改清单
  - 支持导出（PDF/Word/Excel）

  c) 识别结果优化
  - 显示识别置信度（如：准确率 95%）
  - 支持用户反馈（识别错误可标注"不准确"）
  - 标注位置（在图片上框出隐患位置）
  - 关联历史类似隐患

  ---
  2. 作业票智能审核

  产品目标：上传作业票，AI 自动审核，找出问题

  功能描述：

  a) 支持的作业票类型
  - 动火作业票
  - 受限空间作业票
  - 高处作业票
  - 临时用电作业票
  - 吊装作业票
  - 断路作业票

  b) 审核维度
  - 完整性检查：必填项是否填写完整、签字是否齐全
  - 合规性检查：作业时间是否符合规定、安全措施是否齐全
  - 风险识别：识别作业内容中的潜在风险
  - 逻辑一致性：作业内容与安全措施是否匹配

  c) 审核结果
  - 审核结论（通过/不通过）
  - 问题清单（每个问题标注位置、整改建议）
  - 风险提示（关联历史事故案例）
  - 参考案例

  ---
  3. 语音交互

  产品目标：解放双手，现场作业时语音提问

  功能描述：

  a) 语音输入
  - 点击语音按钮，开始录音
  - 支持嘈杂环境识别
  - 支持专业术语识别

  b) 语音输出
  - 回答内容语音播报
  - 可调节语速
  - 支持边听边看（语音 + 文字同步显示）

  c) 典型场景
  - 现场巡检时语音描述隐患
  - 作业过程中语音查询规程
  - 应急情况下语音快速求助

  ---
  4. 隐患上报闭环

  产品目标：发现隐患 → 上报 → 跟踪整改 → 验证闭环

  功能描述：

  a) 上报流程
  - 发现隐患（拍照/语音描述）
  - AI 自动分析并生成报告
  - 用户确认后一键上报
  - 自动推送给责任人

  b) 整改跟踪
  - 整改进度实时查看
  - 超期未整改自动提醒
  - 整改完成后拍照验证
  - 验证通过后自动闭环

  c) 与业务系统联动
  - 隐患数据自动同步到企业 EHS 系统
  - 整改完成后自动通知相关人员

  ---
  技能平台

  1. 知识库构建器

  产品目标：企业可以上传自己的文档，构建专属知识库

  功能描述：
  - 上传企业文档（PDF/Word/Excel）
  - 自动解析和索引
  - 知识库测试（输入问题，查看回答效果）
  - 知识库版本管理

  ---
  2. 更多 API

  产品目标：提供更多 AI 能力

  新增 API：
  - 风险评估 API（输入作业内容，返回风险等级和措施）
  - 应急预案匹配 API（输入事故类型，返回预案内容）
  - 作业票审核 API

  ---
  3. 使用量统计和配额管理

  产品目标：开发者可以清楚了解使用情况

  功能描述：
  - 使用量统计（按 API 分类、按时间统计）
  - 配额管理（免费额度、付费套餐）
  - 调用日志（方便调试）
  - 费用账单
