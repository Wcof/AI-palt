# UI 前景层与按钮颜色优化说明

## 本次调整范围

- 放大 `/ai-app` 会话区抬头：默认标题调整为「极客光年安全智能」，标题字号提升，增强页面识别度。
- 放大底部输入框：输入区域高度、边框、阴影与按钮尺寸同步增强，避免对话框过小、过白、不明显。
- 增强前景层次：会话面板、输入框、消息气泡从纯白调整为白蓝渐变与浅蓝边框，和蓝色安全生产背景形成区分。
- 清理黑色按钮：将残留的黑色主按钮、黑色图标块、黑色弹窗遮罩统一替换为蓝色主按钮、浅蓝图标块或蓝色半透明遮罩。
- 修正报告模板输入区：从黑底 Markdown 改成浅蓝文档输入区。

## 颜色约束

- 主按钮：蓝色 / 蓝青渐变。
- 次按钮：白底 + 浅蓝边框。
- 危险操作：保留红色文字或红色弱提示。
- 不再使用黑色作为主操作按钮颜色。

## 运行说明

```bash
npm install
npm run dev
```

当前包内不包含 `node_modules`，未安装依赖时直接构建会提示 `vue-tsc: not found`。
