# 系统设置

## 页面基础信息
- 路由：`/newAI/system`
- 兼容路由：`/system`

## 页面目标
- 集中承接 License 模板、License 实例、账号授权、模型配置、共创回流、审计日志、用量分析和备份容灾。

## 页面结构
- 全屏页面布局
- 全局左侧边栏在进入系统设置后切换为系统设置菜单，原主导航与会话列表一并隐藏
- 页面内部不再重复放置桌面端左侧 Tab，仅在移动端保留横向菜单
- `版本管理` 主区展示：
  - 版本列表
  - 创建版本弹窗
- `License 管理` 主区展示：
  - 创建 License 弹窗
  - License 列表
- `模型配置` 主区拆为：
  - 带搜索的统一模型列表
  - 新增模型弹窗，弹窗内支持 `第三方模型 / 本地模型` Tab 切换
- `成员与额度` 主区展示：
  - 企业总量控制、额度有效期、已分配额度和超额策略
  - 租户管理列表，支持新增、编辑、删除、启用和停用
  - 企业成员与用户额度分配列表，支持新增、编辑、删除、角色、租户关联、账号状态和用户额度
  - 额度变更记录
  - 6 月资产控制暂缓说明
- `反馈列表` 主区展示：
  - Session ID
  - 反馈人
  - 反馈类型（点赞 / 点踩）
  - 问题摘要
  - 上下文补全（前三轮 + 后一轮）
  - 反馈时间
  - 状态
  - 处置
- `审计日志` 主区展示：
  - 账号
  - 系统
  - IP
  - 浏览器
  - 操作
  - 变更
  - 时间
- `用量分析` 主区展示：
  - 企业累计请求次数
  - 企业累计 Token 用量
  - 使用用户数
  - 合并明细表
    - 第一行固定为“全部”
    - 后续按 Key 展示
    - Key 行支持折叠展开单个 AI 技能明细
  - 时间筛选（今日 / 近 7 天 / 近 30 天）
- `备份与容灾` 主区展示：
  - 快照备份列表
  - 快照时间
  - 该快照包含的服务
  - 手动备份按钮与 30 分钟倒计时
- 不再提供“小程序适配”Tab 和对应内容区

## 字段说明
- 版本管理：版本名称、版本说明、Agent 功能选择、AI 技能功能选择
- License 管理：License 编号、版本、有效期、状态、账号数量、操作
- 模型配置：配置名称、来源、模型协议、Key(Token)、URL(请求地址)、模型名称、测试状态、测试时间、平均耗时
- 额度治理：企业请求上限、企业 Token 上限、存储上限、额度有效期、已分配额度、租户名称、租户编码、负责人、成员数、成员姓名、部门、角色、账号状态、用户请求额度、用户 Token 额度、用户存储额度、额度变更记录
- 共创回流：Session ID、反馈人、反馈类型、问题摘要、完整问题、回答、上下文补全、反馈原因、反馈时间、处理状态
- 审计日志：账号、系统、IP、浏览器、操作、变更、时间
- 用量分析：总请求数、总 Token、总用户数、Key、名称、AI 技能、请求总次数、Token 总用量、时间筛选
- 备份列表：快照编号、快照时间、备份状态、包含服务

## 操作说明
- 新建和维护版本
- 创建 License
- 查看 License 列表
- 启用 / 禁用 License
- 搜索并查看官方本地模型、本地模型与第三方模型
- 通过弹窗新增第三方模型或本地模型
- 测试模型连通性
- 在模型列表中切换默认模型
- 删除自定义第三方模型或本地模型
- 通过系统设置菜单进入 `成员与额度`
- 配置企业额度有效期和企业总次数上限
- 新增、编辑、删除租户
- 新增、编辑、删除企业成员，并给成员显式分配请求次数额度
- 查看额度变更记录
- 按反馈类型和处理状态筛选共创回流，并搜索、悬停查看完整问题、处置状态
- 搜索审计日志
- 查看企业总量和 Key / AI 技能合并明细表
- 点击 `备份当前数据` 触发手动备份

## 交互规则
- 系统设置菜单通过左侧边栏菜单切换，菜单项与 `tab` 路由参数绑定。
- 版本管理和 License 管理拆分为两个独立菜单页面。
- 创建 License 必须先选择版本，再设置有效期。
- 创建 License 使用弹窗，不在页面内直接铺开表单。
- License 列表提供状态列和启用/禁用操作，二者联动更新。
- 创建版本改为弹窗交互，版本管理页面只保留列表和创建入口。
- 创建版本弹窗不再维护“版本编码”和“关联模块”，只保留版本名称、版本说明、Agent 功能选择和 AI 技能功能选择。
- 版本管理按功能选择自动推导模块开通状态：
  - `Agent` 下任意功能被选中，则开通 Agent 模块，并开放所选 Agent 功能；全部取消则不开通 Agent 模块。
  - `AI 技能` 下任意功能被选中，则开通 AI 技能模块，并开放所选 AI 技能功能；全部取消则不开通 AI 技能模块。
  - 选中 `OCR识别` 时，License 功能集同步包含 OCR 能力。
- 创建版本弹窗不提供过期时间字段，该信息仅在创建 License 时维护。
- 企业额度必须配置有效期；有效期外停止 AI 调用。
- 企业请求次数为最高上限；企业成员请求次数额度必须在企业总次数内显式分配，不允许超分配。
- 新增成员默认请求次数额度为 `0`，视为未分配额度；未分配额度的用户不能调用 AI。
- 成员额度变更必须写入额度变更记录，并进入审计日志。
- Token input/output 需要记录并展示，但 6 月不做 Token 余额归零动态熔断。
- 流式预扣费、滚动核销、失败退回和最终消耗属于后续资产控制能力，本期暂缓。
- 模型配置统一使用搜索 + 列表结构，不再显示右侧当前默认模型摘要卡片。
- 官方本地模型为默认内置数据，允许测试和切换，不允许删除。
- 新增模型弹窗必须提供 `第三方模型 / 本地模型` Tab 切换。
- 第三方模型必须填写供应商名称、模型协议、Key(Token)、URL(请求地址)、模型名称后才能保存。
- 本地模型必须填写本地模型名称、模型协议、Token、URL(请求地址)、模型名称后才能保存。
- 第三方模型和本地模型均支持 `OpenAI` 与 `Anthropic` 两种协议。
- 反馈列表不再使用“样本池”口径，反馈必须能回看上下文，并通过“处置”弹窗修改状态。
- 反馈列表支持按类型筛选，类型筛选项为 `全部 / 点赞 / 点踩`。
- 反馈列表支持按状态筛选，状态筛选项为 `全部 / 待处置 / 采纳 / 误判 / 已搁置`，其中 `采纳` 对应处理状态 `已采纳`。
- 反馈列表中的问题列默认显示摘要，Hover 后浮层显示完整问题。
- 处置弹窗中的问答区按内容对话流展示：回答在左、问题在右，并支持上下滚动查看。
- 审计日志与版本台账合并为单一审计日志。
- 用量分析支持时间筛选，并可按 Key 检索调用量。
- 用量分析明细表第一行固定显示 `全部 / 全部 / 全部` 汇总。
- 用量分析明细表中每个 Key 默认显示汇总行，点击后折叠展开该 Key 下实际调用过的 AI 技能明细。
- 手动备份启动后按钮置灰 30 分钟，并提示“正在继续备份操作”。
- 系统设置中的内容展示以列表为主，减少卡片式散点布局。

## 状态流转
- 模板创建后可重复复用。
- License 创建后进入授权管理列表。
- 企业额度有效期内可调用 AI；过期后企业下所有用户停止调用。
- 企业请求次数用尽后，企业下所有用户停止调用；用户请求次数用尽后，仅该用户停止调用。
- 第三方模型或本地模型创建后默认处于“未测试”状态，测试后更新为“测试成功 / 测试失败”。
- 共创回流状态固定为 `待处置 / 已采纳 / 误判 / 已搁置`，通过处置弹窗更新。
- 备份任务启动后，快照状态为 `进行中`；30 分钟后自动转为 `已完成`。

## 异常场景
- 账号未绑定 License 时，未开通功能显示 `未开通 / 需付费`。
- License 不包含某功能时，进入对应页面或执行对应操作弹出 License 提示。
- 企业额度过期、用户停用、未分配请求额度、企业或用户请求次数用尽时，阻止 AI 调用并展示原因。
- 用户额度超分配到企业总额度之外时，阻止保存并提示调整。
- 新增第三方模型或本地模型时，如名称、Token、URL 或模型名称缺失，则阻止保存并提示补全。
- 连通性测试失败时，模型保留在列表中，最近测试状态展示为失败并提示检查协议、URL、Token 或模型名称。
- 备份进行中时，禁止重复点击 `备份当前数据`。

## 权限规则
- 页面权限读取“当前账号绑定的 License 实例最终功能集合”。
- 版本模板只用于复用，不直接参与运行时鉴权。

## 数据规则
- License 模板、License 实例、账号授权、模型配置、共创回流、审计日志、用量分析、备份快照均为本地 mock 数据，使用 Pinia persisted state。

## 验收标准
- 可维护版本模板。
- 可创建版本，并通过 Agent / AI 技能下的功能选择自动开通对应模块。
- 可通过弹窗创建 License，并在列表中查看 License 编号、版本、有效期、状态、账号数量。
- 可通过列表操作启用或禁用 License，状态同步变化。
- 可维护租户列表，支持新增、编辑、删除、启用和停用。
- 可维护企业成员列表，支持新增、编辑、删除、租户关联、角色和账号状态。
- 可在企业总次数内给用户显式分配请求次数额度；未分配额度的用户不能调用 AI；额度变更可追溯。
- 可配置企业额度有效期；过期后停止调用；资产控制暂缓项有明确说明。
- 可按类型（全部/点赞/点踩）和状态（全部/待处置/采纳/误判/已搁置）筛选并搜索共创回流，包含 Session ID、反馈人、问题摘要、反馈类型、时间、状态和处置操作。
- Hover 问题摘要可查看完整问题；处置弹窗可按左右问答流查看完整问题、回答、上下文和反馈原因。
- 审计日志可按账号、系统、浏览器、操作、变更检索，并清晰展示时间。
- 用量分析能展示企业总量，以及 `Key / 名称 / AI 技能 / 请求总次数 / Token 总用量` 的合并明细表，并支持展开 Key 查看技能拆分。
- 可手动启动备份，按钮置灰并显示 30 分钟倒计时。
- 页面中不再展示“小程序适配”入口或清单。
