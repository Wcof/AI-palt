# 项目上下文

## 项目定位
- AI 原型设计与 PRD 同步维护项目。
- 本轮实施目标为：对齐线上 `newAI` 信息架构，并叠加 2026-06 mock 功能。

## 约束
- 不接真实后端服务。
- 不接线上 `geek-ai-web` API。
- 所有能力使用 mock 数据、Pinia 本地状态、假接口闭环。

## 核心范围
- `/newAI/*` 路由与左侧导航对齐。
- AI 助手工作台、知识库、智能体、技能、API Key、系统设置六个主页面。
- 商业化授权、反馈回流、样本池、OCR、工具测试、模型切换、审计、台账、Prometheus、备份、小程序适配。
